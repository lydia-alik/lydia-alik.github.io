Compétence Technique

☕ Java

Ma définition

Java est un langage de programmation orienté objet, utilisé pour développer des applications web, mobiles, desktop et des systèmes. Il est connu pour être portable, robuste et largement utilisé dans le développement logiciel professionnel.

Mes éléments de preuve

J'ai appris Java dès le début de ma formation. J'ai d'abord acquis les bases de la programmation orientée objet, notamment les concepts de classes, d'héritage, d'encapsulation et de polymorphisme, avant de maîtriser progressivement le langage en lui-même.

Au cours de mon cursus scolaire, j'ai travaillé sur plusieurs projets en utilisant Java. Par exemple, lors de la création de mon application SendPack pour mon projet de Licence, j'ai travaillé avec java en structurant l'application avec une architecture claire et une bonne répartition des classes. J'ai veillé à bien séparer les responsabilités entre les différentes couches (interface utilisateur, traitement fonctionnel et les accès aux données). J'ai également mis en place les interactions entre les composants Android (Activities, Fragments, etc.) en assurant une gestion cohérente du cycle de vie. Ensuite, j'ai implémenté la communication avec une base de données MariaDB à travers des requêtes SQL, pour la gestion des données de l'application.

Ensuite, dans le cadre de mon projet de Simulation d'exécution de modèles de procédés logiciels, j'ai approfondi mon utilisation de Java en travaillant sur des problématiques de concurrence. J'ai utilisé le multithreading en m'appuyant sur la classe Thread et l'interface Runnable afin d'exécuter plusieurs tâches en parallèle. Pour gérer les accès concurrents aux ressources partagées, j'ai défini des sections critiques protégées avec le mot-clé synchronized.

J'ai également mis en place des mécanismes de coordination entre threads en utilisant les méthodes wait() et notify(), ce qui m'a permis de contrôler leur exécution et d'éviter les conflits. Cela m'a permis de mieux gérer les problèmes de concurrence.

Dans ce même projet, j'ai utilisé la bibliothèque Swing pour développer l'interface graphique. J'ai appliqué les principes SOLID afin de garantir une bonne maintenabilité du code, en structurant notamment mon application autour de classes bien définies (par exemple exeProduct, exeTask, exeModel…) pour séparer clairement les rôles. J'ai également mis en œuvre le modèle d'architecture MVC (Modèle-Vue-Contrôleur), afin d'améliorer la modularité et l'évolutivité du logiciel.

Enfin, j'ai participé à un projet de développement d'un site de réservation de billets d'avion, dans lequel j'ai utilisé le framework Spring Boot. Ce projet m'a permis de consolider mes compétences en développement web, notamment dans la conception d'API, la gestion des données et l'intégration entre les différentes couches de l'application.

Ces expériences démontrent ma capacité à utiliser Java pour concevoir des solutions fiables et structurées, à appliquer les principes de l'orienté objet, et à manipuler aussi bien des aspects applicatifs que techniques. J'ai su produire un code maintenable et expliquer clairement mes choix de conception lors de mes soutenances.
